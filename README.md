
## GoF 디자인 패턴 (inflear 백기선 강의)

## 학습 목적
- 단순 구현을 넘어서 더 좋은 코드를 만들기 위한 학습.
- 4년차 개발자로서 정체된 성장을 좀 더 개선하기 위한 스터디.
- 개발 전반 메타 지식을 습득
- 다양한 프레임워크에서 사용되는 디자인 패턴에 대한 이해도 향상
